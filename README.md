# Resolução Desafios de TypeScript da Dio

Repositório criado para postar as soluções dos desafios propostos no bootcamp da Dio. As questões estão neste repositório [repositório](https://github.com/lira1705/mentoria-typescript).

Para executar os códigos, primeiramente eles devem ser transpilados, sendo necessário o NodeJS instalado na máquina. Após, instale o TypeScript com o comando abaixo, lembrando que desta forma ele será instalado **globalmente**.

``
npm i --global typescript
``

Feita a instalação para o código seja transpilado, basta executar o código abaixo referente ao arquivo desejado.

``
tsc desafioX.ts
``
